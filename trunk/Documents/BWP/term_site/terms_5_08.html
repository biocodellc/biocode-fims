Barcode of Wildlife Term Definitions

Specimen metadata must be entered in the standard spreadsheet template before it can be uploaded to the BWP Field Information Management System (FIMS).

The most recent version of the spreadsheet template can be downloaded here: BWP_FIMS_Spreadsheet_16_Apr_14.xlsx.

Below are all of the column headings from this spreadsheet, along with detailed definitions and guidance on how to fill out your spreadsheets for this project.

KEY

Required

Not having an entry in a Required column will prompt an Error, and you will not be able to upload your spreadsheet to the FIMS.

Highly Recommended

Not having an entry in a Highly Recommended column will prompt a Warning. You will receive an alert message, but still be able to upload your spreadsheet to the FIMS.

Controlled Vocabulary

All entries in a column with this icon will be checked against a list in a Controlled Vocabulary. If the entry in this column is not in this list, it will either cause an Error or a Warning. Check the Controlled Vocabulary section of the definition to see the list, and to see whether not being in that list will cause an Error or Warning.

processingLab

Category: BWPSpecific

Definition: The lab where the specimen will be processed from extraction through to PCR purification.

Controlled Vocabulary: Warning if not in processingLab List

extractionPlate

Category: MaterialSample

Definition: Barcode of the FluidX plate that is holding the DNA extraction storage vial. This field will not be populated until after the DNA extraction process is complete.

extractionWell

Category: MaterialSample

Definition: This is the plate position of the DNA extraction storage vial. This field will not be populated until after the DNA extraction process is complete.

Controlled Vocabulary: Error if not in plateWell List

voucherID

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/occurrenceID

Definition: This is the unique id given to the specimen the tissue was taken from. It is constructed from the institutionCode, collectionCode (if one exists), and catalogNumber, joined by a colon (:).

Example: USNM:Mammals:1234

institutionCode

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/institutionCode

GenBank Equivalent: Part of specimen_voucher

Definition: This is the acronym for the institution or repository where the specimen voucher is stored. GRBio.org is a registry for all institution codes, and the institutionCode field will be validated against its list of codes.

Example: USNM is the institutionCode for the Smithsonian Natural History Museum (United States National Museum).

Controlled Vocabulary: Warning if not in institutionCode List

catalogNumber

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/catalogNumber

GenBank Equivalent: Part of specimen_voucher

Definition: An identifier (preferably unique) for the record within the data set or collection.

Example: 1234

chainOfCustody

Category: Voucher

Definition: Whether or not a specimen voucher has been collected, transported, and accessioned into a secure repository following chain of custody rules that prevent tampering, contamination, mislabeling or loss. Chain of custody refers to the chronological documentation of the sequence of physical locations of a voucher specimen and the people who had custody of it.

Controlled Vocabulary: Error if not in boolean List

yearCollected

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/year

GenBank Equivalent: part of collection_date

Definition: The four-digit year in which the voucher was collected, according to the Common Era Calendar.

Controlled Vocabulary: Error if not in year List

monthCollected

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/month

GenBank Equivalent: part of collection_date

Definition: The two-digit numerical month in which the voucher was collected. This will be validated to being in the range from 1 to 12.

Example: 2 for a voucher collected in February.

Controlled Vocabulary: Error if not in month List

dayCollected

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/day

GenBank Equivalent: part of collection_date

Definition: The integer day of the month on which the voucher was collected.

Controlled Vocabulary: Error if not in day List

collectedBy

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/recordedBy

GenBank Equivalent: collected_by

Definition: A list (concatenated and separated by semicolons) of names of people, groups, or organizations responsible for collecting the specimen voucher. The primary collector or observer, especially one who applies a personal identifier (recordNumber), should be listed first. The name format should be Given Name, [space], Last Name.

Example: David Schindel

countryOrOcean

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/country

GenBank Equivalent: country

Definition: The country or water body from which the specimen voucher was collected. This field will be validated against the INSDC country list (http://www.insdc.org/country.html).

Controlled Vocabulary: Error if not in country List

scientificName

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/scientificName

GenBank Equivalent: organism

Definition: The full scientific name, with authorship and date information if known. When forming part of an Identification, this should be the name in lowest level taxonomic rank that can be determined.

kingdom

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/kingdom

Definition: The full scientific name of the kingdom in which the specimen voucher is classified.

phylum

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/phylum

Definition: The full scientific name of the phylum in which the specimen voucher is classified.

genus

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/genus

Definition: The full scientific name of the genus in which the taxon is classified.

Example: Puma, "Monoclea"

species

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/specificEpithet

Definition: The name of the first or species epithet of the scientificName of the specimen voucher.

Example: concolor, "gottschei"

tissueBarcode

Category: MaterialSample

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/materialSampleID

Definition: This is the 2D barcode on the FluidX long-term tissue storage vial. This can be read using the tube scanner. This field is required because it is the "primary key" of the spreadsheet. Each tissueBarcode is unique across the BWP project.

sequencingLab

Category: BWPSpecific

Definition: The lab where the purified PCR products will be sequenced.

Controlled Vocabulary: Warning if not in sequencingLab List

tissueRack

Category: MaterialSample

Definition: Barcode of the FluidX rack that is holding the tissue storage vial of the specimen when it enters the lab for processing.

tissuePosition

Category: MaterialSample

Definition: This is the position of the tissue storage vial within the tissue rack when it enters the lab for processing.

Example: tissueWell must be in the format [Letter][2-digit number]. Example: "A01, E07"

tissueType

Category: MaterialSample

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/preparations

Definition: The type of tissue that the DNA was extracted from.

Example: Liver; Muscle; Blood; Leaf; Stem; Seed

collectionCode

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/collectionCode

GenBank Equivalent: Part of specimen_voucher

Definition: The name or acronym identifying the collection or data set (from the institution/repository listed in institutionCode) from which the record was derived. This collectionCode should be registered, along with the institutionCode, in GRBio.org.

Example: Mammals

decimalLatitude

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/decimalLatitude

GenBank Equivalent: part of lat_lon

Definition: The geographic latitude,in decimal degrees of the geographic center of a Location. Positive values are north of the Equator, negative values are south of it. This field will be validated to being within the range -90.0 to 90.0.

decimalLongitude

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/decimalLongitude

GenBank Equivalent: part of lat_lon

Definition: The geographic longitude,in decimal degrees of the geographic center of a Location. Positive values are east of the Greenwich Meridian, negative values are west of it. This field will be validated to being within the range -180.0 to 180.0.

identifiedBy

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/identifiedBy

GenBank Equivalent: identified_by

Definition: A list (concatenated by semicolons) of names of people who assigned the Taxon to the specimen voucher. The name format should be Given Name and Last Name.

Example: David Schindel

yearIdentified

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/dateIdentified

Definition: The 4-digit year in which the specimen voucher was identified as representing the Taxon, according to the Common Era Calendar.

Controlled Vocabulary: Warning if not in year List

monthIdentified

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/dateIdentified

Definition: The two-digit numerical month in which the specimen voucher was identified as representing the Taxon. This will be validated to being in the range from 1 to 12.

Example: 2 for a voucher identified in February.

Controlled Vocabulary: Error if not in month List

dayIdentified

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/dateIdentified

Definition: The integer day of the month on which the specimen voucher was identified as representing the Taxon. This will be validated to being in the range from 1 to 31.

Controlled Vocabulary: Error if not in day List

typeStatus

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/typeStatus

GenBank Equivalent: type_material

Definition: A list (concatenated by semicolons) of nomenclatural types (type status, typified scientific name, publication) applied to the subject.

Example: DarwinCore recommends a controlled vocabulary of holotype, paratype, neotype, syntype, lectotype, paralectotype, hapantotype, other.

Controlled Vocabulary: Error if not in typeStatus List

basisOfID

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/identificationReferences

Definition: Free-text field describing what rationale was used to identify the specimen voucher as representing the Taxon. Can include materials examined or reference specimens, literature, or field guides.

identificationConfidence

Category: Identification

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/identificationVerificationStatus

Definition: The subjective appraisal by the identifier of the reliability of the identification at the species level. The reliability of a species identification can be High, Medium or Preliminary, based on the identifier’s expertise in the taxonomic group, the nature and quality of the specimen, and the reference material used to make the identification.

class

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/class

Definition: The full scientific name of the class in which the specimen voucher is classified.

order

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/order

Definition: The full scientific name of the order in which the specimen voucher is classified.

Example: Carnivora, "Monocleales"

family

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/family

Definition: The full scientific name of the family in which the specimen voucher is classified.

preservative

Category: MaterialSample

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/preparations

Definition: List exactly what the tissue is preserved in in the tissue storage vial.

Example: 80% EtOH, 95% EtOH, buffer

extractionBarcode

Category: MaterialSample

Definition: This is the 2D barcode of the FluidX storage tube which contains the DNA extract of the specimen. This field will not be populated until after the DNA extraction process is complete.

sex

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/sex

GenBank Equivalent: sex

Definition: The sex of the specimen voucher.

Example: DarwinCore recommends a controlled vocabulary of unknowable, undetermined, female, male, hermaphrodite, gynandromorph.

Controlled Vocabulary: Error if not in sex List

lifeStage

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/lifeStage

Definition: The age class or life stage of the organism when it was vouchered.

Example: DarwinCore recommends a controlled vocabulary of zygote, embryo, larva, juvenile, adult, sporophyte, spore, gametophyte, gamete, pupa.

Controlled Vocabulary: Warning if not in lifeStage List

associatedMedia

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/associatedMedia

Definition: A list, concatenated by semicolons, of media associated with the specimen voucher such as images or videos.

Example: http://flic.kr/p/746gTw, "http://youtu.be/yjoU6M3nnMA"

establishmentMeans

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/establishmentMeans

Definition: The process by which the biological individual represented in the specimen voucher became established at the location.

Example: DarwinCore recommends a controlled vocabulary of native, introduced, naturalised, invasive, managed, uncertain.

Controlled Vocabulary: Warning if not in establishmentMeans List

sampleRemarks

Category: Voucher

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/occurrenceRemarks

Definition: Comments or notes about the specimen voucher that are not covered in the other fields.

eventID

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/eventID

Definition: An identifier for the set of information associated with an Event (something that occurs at a place and time). May be a global unique identifier or an identifier specific to the data set.

collectionMethod

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/samplingProtocol

Definition: The name of, reference to, or description of the method or protocol used during an Event.

Example: UV light trap, "mist net", "ad hoc observation", "Takats et al. 2001. Guidelines for Nocturnal Owl Monitoring in North America. Beaverhill Bird Observatory and Bird Studies Canada, Edmonton, Alberta. 32 pp.", "http://www.bsc-eoc.org/download/Owl.pdf".

fieldNumber

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/fieldNumber

Definition: An identifier given to the individual in the field. Often serves as a link between field notes and the Event.

timeCollected

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/eventTime

Definition: The time or interval during which a voucher was collected. Can be a descriptive term, such as "morning". If using an exact time of day, use the 24-hour time format (e.g "1400" for 2 hours past noon).

permitInformation

Category: Event

Definition: A free-text field containing information about the collecting permit used to collect the specimen voucher. It can include a permit number or a link to the permit itself.

habitat

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/habitat

Definition: A category or description of the habitat in which the Event occurred.

Example: oak savanna, "pre-cordilleran steppe"

eventRemarks

Category: Event

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/eventRemarks

Definition: Comments or notes about the collecting event that are not covered in the other fields.

waterBody

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/waterBody

Definition: The name of the water body in which the Location occurs. Recommended best practice is to use a controlled vocabulary such as the Getty Thesaurus of Geographic Names.

Example: Indian Ocean, "Baltic Sea", "Hudson River"

island

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/island

GenBank Equivalent: part of country

Definition: The name of the island on or near which the Location occurs.

Example: Isla Victoria, "Vancouver", "Viti Levu", "Zanzibar"

stateProvince

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/stateProvince

GenBank Equivalent: part of country

Definition: The name of the next smaller administrative region than country (state, province, canton, department, region, etc.) in which the Location occurs.

Example: Montana, "Minas Gerais", "Córdoba"

county

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/county

GenBank Equivalent: part of country

Definition: The full, unabbreviated name of the next smaller administrative region than stateProvince (county, shire, department, etc.) in which the Location occurs.

Example: Missoula, "Los Lagos", "Mataró"

municipality

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/municipality

GenBank Equivalent: part of country

Definition: The full, unabbreviated name of the next smaller administrative region than county (city, municipality, etc.) in which the Location occurs. Do not use this term for a nearby named place that does not contain the actual location.

locality

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/locality

GenBank Equivalent: part of country

Definition: The specific description of the place. Less specific geographic information can be provided in other geographic terms (higherGeography, continent, country, stateProvince, county, municipality, waterBody, island, islandGroup). This term may contain information modified from the original to correct perceived errors or standardize the description.

Example: Bariloche, 25 km NNE via Ruta Nacional 40 (=Ruta 237)

verbatimElevation

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/verbatimElevation

Definition: The original description of the elevation (altitude above sea level) of the Location. This value should be given in meters.

Example: 100-200 m

verbatimDepth

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/verbatimDepth

Definition: The original description of the depth below the local surface. This value should be given in meters.

Example: 5-10 m

locationRemarks

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/locationRemarks

Definition: Comments or notes about the location of the collecting event that are not covered in the other fields.

Example: under water since 2005

georeferenceProtocol

Category: Location

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/georeferenceProtocol

Definition: A description or reference to the methods used to determine the spatial footprint, coordinates, and uncertainties.

Example: Guide to Best Practices for Georeferencing (Chapman and Wieczorek, eds. 2006), Global Biodiversity Information Facility.", "MaNIS/HerpNet/ORNIS Georeferencing Guidelines", "Georeferencing Quick Reference Guide"

subfamily

Category: Taxon

Definition: This is the full scientific name of the rank immediately below family but above genus of the specimen voucher.

subspecies

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/infraspecificEpithet

Definition: The name of the lowest or terminal infraspecific epithet of the scientificName of the specimen voucher, excluding any rank designation.

Example: concolor, "oxyadenia", "sayi"

commonName

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/vernacularName

Definition: A common or vernacular name.

Example: Andean Condor, "Condor Andino", "American Eagle", "Gänsegeier"

associatedTaxa

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/associatedTaxa

Definition: A list (concatenated and separated by semicolons) of names of taxa and their associations with the specimen voucher.

Example: host: Quercus alba

taxonRemarks

Category: Taxon

DarwinCore Equivalent: http://rs.tdwg.org/dwc/terms/taxonRemarks

Definition: Comments of notes about the taxon or name.

Example: this name is a misspelling in common use
