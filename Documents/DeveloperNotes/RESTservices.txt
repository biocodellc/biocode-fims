

 -c,--configuration <arg>      Configuration File
 -d,--demo_mode                Demonstration mode.  Do not need to specify
                               expedition_code, configuration, or
                               output_directory.  You still need to
                               specify an input file.
 -h,--help                     print this help message and exit
 -i,--input_file <arg>         Input Spreadsheet
 -o,--output_directory <arg>   Output Directory
 -p,--expedition_code <arg>       Expedition code.  You will need to obtain a
                               expedition code before loading data, or use
                               the demo_mode.
 -t,--triplify                 Triplify
 -u,--upload                   Upload
 -w,--write_spreadsheet        Write back an excel spreadsheet from the
                               triplestore.  This option is useful for
                               testing output from flatfile -> RDF ->
                               flatfile but not necessary.
